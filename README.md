## Run it

### Run local

Clone it, create a .env file folowing the .example.env, run redis locally, npm install and:

```bash
npm run start:tsnode
```

or

```bash
mpm start
```

### Run with docker:

Clone it, create a .env file folowing the .example.docker.env, and:

```bash
docker-compose build
docker-compose up
```
